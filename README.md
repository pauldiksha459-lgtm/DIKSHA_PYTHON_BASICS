# 🐍 Basics of Python

- 🔁 Conditional Statements (if-else)  
- 🔄 Loops (for, while)  
- 📦 Functions  
- 📋 Lists, Tuples, Sets, Dictionaries  
- 🔍 Basic Input and Output
